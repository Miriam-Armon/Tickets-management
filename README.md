# Tickets-management
Full-stack ticket management application (home assignment).

- **Backend**:
   ASP.NET Core Web API (`TicketsManagement.Api`)  
- **Frontend**:
   Angular (`tickets-client`)

## Getting Started

1. **Clone the repository**  

git clone https://github.com/YourUsername/Tickets-management.git  
cd Tickets-management  

2. **Backend setup**  
cd TicketsManagement.Api  
dotnet restore  
dotnet run  

3. **Fronted setup**  
cd ../tickets-client  
npm install  
ng serve --open  

**Angular app will run at http://localhost:4200 and connect to the backend automatically**
